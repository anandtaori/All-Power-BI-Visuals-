# 📊 Power BI Data Visualization Project

This repository contains data visualizations created using **Power BI** from a sales dataset provided in an Excel file. The project demonstrates different chart types used to analyze sales, profit, regions, categories, and customer segments.

All visuals generated from the dataset are compiled in **Chart.pdf** for easy viewing.

---

## 📁 Files in this Repository

- **DV Sales Data.xlsx** – Source dataset used for creating the visualizations  
- **Chart.pdf** – Collection of all Power BI charts and dashboards

---

## 📈 Visualizations Included

This project includes the following Power BI visuals:

- Stacked Bar Chart  
- Stacked Column Chart  
- Pie Chart  
- Donut Chart  
- Clustered Bar Chart  
- Clustered Column Chart  
- Line Chart  
- Area Chart  
- Ribbon Chart  
- Scatter Chart (Sales vs Profit)  
- Map Visualization (Sales by City)  
- Table  
- Matrix  
- Funnel Chart  
- Gauge Chart  
- KPI Indicator  
- Decomposition Tree  
- Treemap

---

## 🎯 Project Objective

The purpose of this project is to practice and showcase **data visualization techniques in Power BI** using a sample sales dataset.

## 🛠 Tools Used

- **Power BI Desktop**
- **Microsoft Excel**
